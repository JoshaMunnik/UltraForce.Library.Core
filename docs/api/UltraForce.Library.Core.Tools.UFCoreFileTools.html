CheckAlternativeFile(string, string, string)

Checks if there is a file using an alternative extension, if there is the method will copy the alternative file to the missing file.

This method uses File.Copy but does not handle any of the exceptions it can throw. 

public static bool CheckAlternativeFile(string missingFileName, string requiredExtension, string alternativeExtension)
