A base class for models in a data service that encapsulates an entity model from a database.

The default implementation of the CopyFromEntityAsync(TEntity) and CopyToEntityAsync(TEntity, UFEntityAction) uses reflection and copies all properties of the service model class that have been annotated with UFEntityAttribute. The code caches property info mapping (so for every model type the properties are only scanned once), but for optimal speed subclasses can override the CopyToEntityAsync(TEntity, UFEntityAction) and CopyFromEntityAsync(TEntity) methods to provide a faster implementation.
